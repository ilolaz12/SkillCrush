Hi, my name is Natalie and I have some advice for you!

Learn actively

Retain what you learn by coding along with video coding tutorials you watch, or by quizzing yourself on what you have learnt.

Learn from a wide variety of sources

Here are some ideas:

  • Sign up for coding email newsletters such as CSS Tricks and Learn to Code With Me.
  • Set your browser's homepage to daily.dev for all the latest developer news.
  • Follow coding teachers and influencers on LinkedIn, Instagram and Twitter.

Make good use of your free time

Listen to coding podcasts when doing the washing up or while travelling by train or bus.

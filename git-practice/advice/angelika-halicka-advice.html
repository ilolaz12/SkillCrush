Hi, my name is Angelika and I have some advice for you!

Believe in yourself

Believe in what you can do. Learning new skills is always hard but as long as you keep going an practicing it will get easier over time. So keep going and don't give up!

Understand one concept before moving on to the next

Make sure that you understand the concepts. You don't necessarily need to remember all the syntax, however understanding each concept is important so that you can keep building on your existing knowledge.

Ask for help

Don't be afraid to ask for help if you're not sure about something. Nobody knows everything and if you ask, you will know for the next time.

Try out different things and practice what you already know

Experiment with different styling, add something to your web page that you maybe haven't done before, experiment. Practice the skills you already have so you can improve and build on them.

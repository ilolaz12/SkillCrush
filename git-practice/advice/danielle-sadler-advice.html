Hi, my name is Danielle and I have some advice for you!

Do a little bit of programming every day!

Coding is learning another language, and learning any language takes practice. In order to get more comfortable using the language, set up the habit of practicing the language of coding every day. This will mean being uncomfortable because you are going to make mistakes. You are also getting in the habit of being uncomfortable every day while keeping the goal in sight of getting more more and more comfortable with coding.

Start to think of yourself as a programmer.

What are the characteristics that you want to embody as you move into tech? Start to be those things before you're ready. Read articles, follow people in technology, use affirmations - find the things that help you see yourself in the process of achieving the goals you want to achieve by starting a tech career.

Ask for help.

Find the resources that work best for you to get help, with your coursework and in working through your fears. Join or create a positive community that can support the amazing things that you're doing!
